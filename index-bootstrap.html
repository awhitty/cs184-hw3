CS 184 Style Guide with Bootstrap

Page organization

The page is divided into three components: header, #main, and footer. Each component has a .container. The .container class provides a basic responsive layout to re-scale some of the content.

Sections

Each division of this guide is organized into sections. There are no grids in the Bootstrap-less version, so all content should just be split up in large sections and styled individually. Section titles should range from h2 to h6 and can contain .muted subtitles. Each section (except the last one) has a 1 pixel border on the bottom to separate the content.

Elements

All HTML elements should render pretty nicely, following the minimalist aesthetic provided by the Skeleton CSS framework and my own modifications, but I'll go over a few to demonstrate it.
